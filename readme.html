Church WordPress Project

WordPress проект для командной разработки. Локально работает через DDEV, фронтенд на Vite, тема кастомная.

1. Локальная установка

  1. Сначала обновляем систему: 
    sudo apt update && sudo apt upgrade -y
  2. Проверяем установку Node, npm и git: 
    node --version
npm --version
git --version
  3. Если Node ниже 22 версии — обновляем.
  4. Устанавливаем DDEV: 
    curl -fsSL https://raw.githubusercontent.com/ddev/ddev/master/scripts/install_ddev.sh | bash
  5. Проверяем установку: 
    ddev version
  6. Клонируем проект: 
    git clone https://github.com/dekobad228-web/Church.git
cd Church
  7. Запускаем окружение: 
    ddev start
  8. Устанавливаем PHP зависимости: 
    ddev composer install

2. Фронтенд (Vite)

  1. Переходим в тему: 
    cd wp-content/themes/main
  2. Устанавливаем зависимости: 
    npm install
  3. Запускаем dev сервер: 
    npm run dev

3. База данных

  1. Если есть файл базы: 
    ddev import-db --file=db.sql

4. Запуск проекта

  1. Открыть сайт командой: 
    ddev launch
  2. Или по адресу: 
    https://church.ddev.site

5. Структура проекта

  1. .ddev — окружение
  2. wp-admin и wp-includes — ядро WordPress
  3. wp-content/themes/main — основная тема
  4. src — исходники фронтенда
  5. dist — сборка фронтенда
  6. PHP логика темы
  7. Vite конфигурация
  8. package.json
  9. db.sql — база данных

6. Основные команды

  1. DDEV: 
    ddev start
ddev stop
ddev restart
ddev launch
ddev import-db --file=db.sql
ddev export-db --file=db.sql
  2. Frontend: 
    npm run dev
npm run build
  3. PHP: 
    ddev composer install

7. Workflow разработки

  1. git pull
  2. ddev start
  3. npm install (если нужно)
  4. npm run dev
  5. git add .
  6. git commit -m "update"
  7. git push

8. Исключения из Git

  1. node_modules
  2. vendor
  3. wp-config.php
  4. .env файлы
  5. uploads

9. Важно

  1. Каждый разработчик использует свой локальный конфиг через DDEV
  2. Проект полностью воспроизводим
  3. Разворачивается через: клонирование, ddev start, установка зависимостей, импорт базы, запуск Vite
